## Leçons apprises
- ceifa/garrysmod (pas ceifa/garrysmod-docker) est le bon nom d'image
- Ne PAS monter un bind mount vide sur /home/gmod/server → ça écrase srcds_run
- Utiliser un named volume Docker pour persister les fichiers serveur
- SteamCMD re-télécharge ~3.8GB à chaque restart sans volume persistant
- En mode debug (-gdb), les logs srcds ne vont pas dans Docker logs
- DarkRP n'est PAS inclus dans l'image de base → il faut l'installer
